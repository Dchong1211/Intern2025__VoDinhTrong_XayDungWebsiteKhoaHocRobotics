CÁC CHỨC NĂNG CHÍNH CỦA WEB
- Dùng dể admin có thể tạo tài khoản và cấp mật khẩu cho các giáo viên, admin có chức năng quên mật khẩu phòng trường hợp admin quên mật khẩu của
mình bàng cách nhấn quên mật khẩu và nó sẽ guiwr mã gồm 6 chữ số tới gmail admin
- Admin có các chức năng chính như xem, thêm, sửa, ẩn hiện, phân quyền các chương trình cũng như các khóa học và bài giảng
- Admin có quyền phân công cho giáo viên nào dạy khóa học nào cũng như có thể khóa tài khoản giáo viên hoặc ẩn khóa học, bài học
- Giáo viên được nhận tài khoản mật khẩu từ admin và sử dụng. Trong đó giáo viên chỉ được quyền xem những chương trình, khóa học, bài giảng mà mình
đã được phân công thôi còn những bài giảng không được phân công thì không thể nhìn thấy.

HƯỚNG DẪN CHẠY SOURCE CODE
- Nếu lần đầu lấy code về nên vào thư mục chính Intern2025__VoDinhTrong_XayDungWebsiteKhoaHocRobotics để chạy câu lệnh "npm install" 
trong terminal để có thể lấy toàn bộ thư viện node về vì GitHub không tự tải các thư mục đó lên vì quá nặng.

Cách 1: 
  - Sau đó vào tạo terminal mới từ Intern2025__VoDinhTrong_XayDungWebsiteKhoaHocRobotics
  - Nhập lần lượt các câu lệnh "cd backend" "npm run dev"
  - Sau đó tiếp tục mở terminal mới từ Intern2025__VoDinhTrong_XayDungWebsiteKhoaHocRobotics
  - Nhập lần lượt các câu lệnh "cd frontend" "npm run dev"
  - Sau đó Ctrl + chuột trái vào link  http://localhost:5173/ để mở đến trang đăng nhập web

Cách 2:
  - Hoặc đơn giản hơn cách trên là chỉ cần chạy file run.bat sau đó Ctrl + chuột trái vào link  http://localhost:5173/ để mở đến trang đăng nhập web
